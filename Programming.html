Data Structures

This was my first actual programming class. I learned what data structures are, how to use them efficiently, and how to build them while also learning about object oriented languages (we used Java). My first challenging project was creating a circular doubly linked list. A circular doubly linked list is a linked list that starts with a handle node and is connected to two other nodes called previous and next. Each of those nodes have a previous and next node next them. This forms a circle, and because it is a circle you can traverse the linked list clockwise or counter-clockwise, allowing it to be faster than a conventional linked list which can only go to the next node. Algorithms

Algorithms was my first challenging theory heavy based course in my journey to learn computer science. In this class I learned how to find the best case, worst case, and average case of algorithms. In order to get us to learn more about sorting algorithms we had a competition with our class where we were tasked to create a sorting algorithm with Java to sort a list integers based on three criteria. My team (there were 18 teams of two people each) placed third by using MergeSort, a modified version of Bucket Sort, and another different modification of Bucket Sort. We Also learned about Turing Machines, DFAs, NFAs,and graph theory
